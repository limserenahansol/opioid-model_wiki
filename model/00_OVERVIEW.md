## One-sentence model

Opioid seeking on PR is generated by a **low-dimensional latent motivational process** that integrates reward, failure, pause, and context — with **distinct update architectures** for contingent (active) versus non-contingent (passive) opioid history.

---

## Implementation ladder

```
M0  single latent engagement x_t  (drift + noise)
M1  x_t → observable lick output λ_t
M2  dual axes V_t (reward/value) and D_t (deficit/withdrawal)
M3  passive Pavlovian context C_t × withdrawal gain G_t
M4  pause / re-engagement extension
```

Fit and compare on **lick-level data** using simulation-based inference (SBI).
