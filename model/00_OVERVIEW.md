## One-sentence model

Opioid seeking on PR is generated by a **low-dimensional latent motivational process** that integrates reward, failure, pause, and context — with **distinct update architectures** for contingent (active) versus non-contingent (passive) opioid history.

## Implementation ladder

```
Tier 1 (now):  M0 drift  →  M1 λ=f(x)
Tier 2:        per-event r vs smoothed E[r|T]
Tier 3 (if ID): M2  V_t, D_t  (+ interaction)
Tier 4 (if ID): M3b C_t × G  (generalized motivation; not literal PIT)
Later:         M4 pause · pupil · neural
```
